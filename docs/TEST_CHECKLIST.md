# Test Checklist

Khi ban da co data that, check theo danh sach nay:

## Functional

- Script co doc duoc khong
- Script co tach thanh segment hop ly khong
- Moi segment co keyword, scene, tone khong
- `analysis_requests.json` co dung provider/model mong muon khong
- Moi segment co audio path khong
- Moi segment co image path khong
- Timeline co start/end time dung thu tu khong

## TTS

- Audio co tao that khong
- Giong doc co nghe tu nhien khong
- Duration co khop noi dung khong

## Image retrieval

- Anh co hop chu de khong
- Anh co hop tone khong
- Anh co bi lap lai qua nhieu khong

## Resolve integration

- Media co import duoc khong
- Timeline co tao duoc khong
- Audio va image co nam dung vi tri khong

## Demo quality

- Toan bo pipeline co chay tu dau den cuoi khong
- Co it nhat 1 script demo on dinh khong
- Co the quay video demo trong 1 lan chay khong
